📖 PrusaSlicer Setup Guide
1 Correct Printer & Material Settings
✔ Select: Prusa CORE One HF0.4
✔ Select: Prusament PLA
Printer: Must be Prusa CORE One HF0.4 nozzle. Filament: Must be Prusament PLA (or your approved material). These settings are required for accurate billing.
2 Where to Find Billing Values
Filament weight (g) → first number
Print time (normal mode)
After slicing, find the Sliced Info panel. Use Used Filament (g) — the first number before parentheses — and Estimated printing time in normal mode.
G-Code File Naming — Required

When using the Prusa printers, you must rename your G-code file with the prefix of your JHED ID and your Senior Design team name before sending it to the printer.

cgunthe3_GuntherAerospace_PartName.gcode
Format: <JHED>_<TeamName>_<description>.gcode

This is required for accurate billing. Prints without proper naming may be removed from the queue or billed incorrectly.
